Meet Olya AI ✨

Your highly advanced WhatsApp Personal Assistant. Perfectly crafted by Hansaka P. Fernando. Equipped with Interactive Menus, Real-time AI Generation, ECCPMS Systems, and Next-Gen Interface.

Explore Powers

Core Capabilities 💎

AI Image Generator

Describe your imagination and Olya will paint it instantly. Integrated with Pollinations AI for stunning 1024x1024 visual masterpieces with fully interactive response buttons.

Interactive Downloader

Say goodbye to typing numbers. Send a YouTube/FB link and select 🎧 Audio, 🎥 Video, or 📂 Document directly via modern WhatsApp UI buttons.

ECCPMS PDF Reports

For Prefect Management: Scan barcode images via Vision AI, verify via secure Firestore Database, and generate digitally signed custom PDF Reports directly in chat.

Smart Conversational AI

Natural intelligence intent detection. She knows when to say "Thank You", when to alert the master, and she communicates entirely with natural Sinhala templates.

Photo-to-Text (OCR)

Send a photo containing text, and Olya will meticulously read it and extract all the characters into a neat text message. Perfect for copying notes!

Anti-Delete & Security

No message escapes. If a user deletes a message, Olya intercepts it, restores it with a stylish note, and maintains strict conversational surveillance.

Command Reference 🛠️

Command Alias / Quick Use Description
.menu None Opens the interactive Main Menu with categorized lists.
.imagine .img / .draw Generates AI Art from text prompt. (e.g. .imagine a beautiful sunset)
.read .ocr / .text Reply to an image with this to extract all text from it.
.yt .song / .video Searches or Downloads YouTube/FB Videos. Returns Interactive Buttons.
.movie .film Fetches IMDb ratings, plot, cast, and a generated movie poster.
.search .google Asks Olya AI to intelligently search the web and send bullet-point answers.
.tts .speak Converts the provided text into a playable Voice Note (Audio).
.alive .ping / .status Checks the Bot's online status, system uptime, and fetches the Dev info.
